#1 HTML Practice

For this project, I followed a LinkedIn Learning Course to guide me through creating a digital project page for a terrarium. I started by learning HTML, which helped me structure the webpage and add headings, paragraphs, and images. The course guided me through HTML basics like:

  • Set up the basic HTML structure ‚Äì using DOCTYPE html to indicate HTML5, html lang="en" for accessibility and SEO, and head and body sections to separate metadata from visible content.
  • Add a webpage title and meta tags ‚Äì setting the title defined the tab name, while meta tags ensured my website supported international characters and scaled properly across devices.
  • Use headings, paragraphs, and divisions ‚Äì h1> through h3> helped structure the page, while p> and div> allowed me to organise content into sections like the terrarium containers and plant hold.
  • Assign IDs and classes ‚Äì IDs (like #left-container) and classes (like .plant-holder) let me target specific elements in CSS and JavaScript for styling and interactivity.
  • Embed images and videos ‚Äì using img> and video> tags, I added visuals to enhance understanding and engagement. Alt attributes ensured accessibility and fallback descriptions.

#2 CSS Practice

  • Apply basic styling to the webpage ‚Äì setting the font family, text colours, and alignment for headings and paragraphs. For example, I made the main title brown and centred it for better readability.
  • Use IDs and classes for styling ‚Äì IDs like #left-container allowed me to style specific sections differently, while classes like .container or .plant-holder let me apply consistent styles across multiple elements.
  • Style containers and panels ‚Äì I created left and right panels to display plant images, assigning widths, background colours, and padding to organise content neatly.
  • Add interactive hover effects ‚Äì for instance, hovering over the main heading or plant images triggered a colour change or slight enlargement, adding dynamic interactivity.
  • Create animations ‚Äì using @keyframes, I made the plants slide in when the page loaded, enhancing the user experience.
  • Design the terrarium visually ‚Äì I styled the jar walls, top, bottom, soil, and glossy highlights to resemble a real terrarium. I learned to use position: absolute, z-index, and opacity to layer elements realistically.

#3 JavaScript Practice

  • Understanding the Document Object Model (DOM) ‚Äì I learned that the DOM represents the webpage as a tree of objects, and JavaScript can manipulate these objects to update content dynamically.
  • Adding event listeners ‚Äì I used event listeners to detect clicks, key presses, and other actions. For example, clicking a plant or dragging an object triggered specific JavaScript functions.
  • Creating functions ‚Äì Functions grouped instructions into reusable blocks of code, allowing me to perform tasks like updating the terrarium or animating elements when called.
  • Using variables ‚Äì I applied let for values that could change, const for fixed values, and understood when var could be used for older syntax.
  • Making the webpage interactive ‚Äì I learned to respond to user actions, such as dragging plants, triggering animations, and dynamically updating the display without reloading the page.
  • Debugging and problem-solving ‚Äì JavaScript often produced errors at first, so I learned to carefully read error messages, test small code segments, and look up solutions on forums like StackOverflow.
